#### 接口文档

简化版的商城API接口:http://api.cat-shop.penkuoer.com

- 接口代理

为第三方api接口调用提供代理服务

```
url
  /api/v2/proxy
method
  post
params
  url   需要请求的api接口地址
```

- 文件上传

```
url
  /api/v1/common/file_upload
method
  post
params
  file

返回数据
{
  "code": "success",
  "message": "文件上传成功",
  "info": "/uploads/20190221/1550738827910.jpeg"
}
```
